# Sentiment Analysis in Stock Market Prediction Using LlaMA 2

## About
This project, as part of a Bachelor's thesis, explores sentiment analysis using Meta's LlaMA 2 for predicting stock returns, particularly in the OMXS30 index, demonstrating the integration of NLP in financial markets.

## Features
- Sentiment analysis with LlaMA 2.
- Stock price data processing.
- Market prediction using Random Forest.
- Portfolio construction and evaluation.
